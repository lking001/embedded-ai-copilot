# embedded-ai-copilot
在嵌入式开发（如 STM32 / FreeRTOS / DMA 驱动开发）过程中，开发者需要频繁查阅手册、调试寄存器、分析时序问题，效率较低。因此我构建了一个基于 AI Agent 的开发辅助系统，用于提升开发效率。
# Embedded AI Copilot

基于多 Agent 协作的嵌入式开发智能助手，用于提升 STM32 / FreeRTOS 开发效率。

## Features

- 📄 文档解析 Agent（RAG）
- ⚙️ 驱动生成 Agent
- 🐞 调试分析 Agent
- 🧠 多 Agent 协作调度
